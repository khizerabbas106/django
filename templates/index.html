This Website is designed for display some of the most suseptible lakes in Hunza River Basin using machine learning. The main objective of this website is Modelling of GLOF Susceptibility and Assessment of Potential Lake Outbursts using Machine Learning and Remote Sensing Datasets.

What is GLOF?

“Glacial Lake Outburst Flood” (GLOF) is a phrase used to describe a sudden release of a significant amount of water retained in a glacial lake. GLOFs have the potential to cause significant amount of damage and destruction to lives, infrastructure, and environment in nearby regions.
